User Guide & FAQ

Welcome to the Vulnerability Scanner Dashboard! This tool helps you audit systems on your network for open ports and known security vulnerabilities.

1. How do I start a scan?

  1. Go to the Dashboard page.
  2. Enter target IP addresses, subnets, or hostnames (e.g. 192.168.1.15, scanme.nmap.org) separated by commas.
  3. Toggle options:
    • Fast Scan: Scans the top 100 common ports (saves time).
    • OS Detection: Detects system operating system types.
    • HTTP Web Audit: Scans web services for SQL injection, XSS, and SSL vulnerabilities.
  4. Click Launch Scan. An active blue banner at the top will indicate scan progress.

2. How do I view results?

Go to the Scan History tab. Click Open to view individual host records and vulnerabilities, or click Download HTML to save a detailed, client-ready printable audit report.
